Overview

Purpose

My purpose of this website is a way for me to keep track of daily progression for reflection. I can keep track of what I do each day so I know what I have completed and need to work on or to work harder or work less.

Audience

My audience is mostly myself as this is more like a personal progress or journal website. I could also show this websites to others if I want to show people my daily progress/ routine.

Dynamic elements

I won't have too much javascript but I like dynmically organizing elements as well as using javascript objects. So I am going to have an object of jounral entries. I can display them as well as filter and search for specific dates.
